## Theme Frontmatter

| Field | Type | Description |
|-------|------|-------------|
| status | string | in_progress, hold, completed |
| page_type | string | Always "theme" |

## Structure

Themes are ongoing strategic directions. They live in `21 Themes/` and group related goals.

## Hierarchy

```
Vision → Theme → Objective → Goal → Task
```

Themes are long-lived — they represent areas of focus, not time-bound deliverables.
