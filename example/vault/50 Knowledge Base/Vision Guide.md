## Vision Frontmatter

| Field | Type | Description |
|-------|------|-------------|
| status | string | in_progress, completed |
| page_type | string | Always "vision" |

## Structure

Visions live in `20 Vision/` and represent long-term aspirations. Themes align to a vision.

## Hierarchy

```
Vision → Theme → Objective → Goal → Task
```

A vault typically has 1-3 active visions.
