   - If `scopeDir` is set → set `CONTENT_PATH=filepath.Join(vaultPath, scopeDir)`
   - Run `semantic-search-mcp search <query>` as subprocess with `CONTENT_PATH` env var
   - Print results to stdout
   - Return error if subprocess fails or `semantic-search-mcp` not found

   - Searches entire vault (no scopeDir)

3. Add `search "query"` subcommand to each type group (task, goal, theme, objective, vision):
   - Same `--top-k` flag
   - Passes the type's directory as scopeDir

4. Create `./pkg/ops/search_test.go` with tests using counterfeiter or exec mocking

5. Add counterfeiter generate comment for SearchOperation interface
