<objective>
Add --assignee flag to all list commands so users can filter tasks/goals/etc by the assignee frontmatter field.
Example: `vault-cli task list --assignee bborbe` or `vault-cli task list --assignee localclaw`
</objective>

<context>
Go CLI project at ~/Documents/workspaces/vault-cli.
Tasks already have an `assignee` field in frontmatter (e.g. `assignee: bborbe`).
The list command currently supports --status and --all filters.
Read CLAUDE.md for project conventions.
</context>
